To add phrases to the game, go categories, click on the file to add the phrase to, and place each phrase on a new line. Individual words on the same line are counted as part of the same phrase.

Only use letters in your phrases, no commas, apostrophes, quotes, etc.

To create topics, create a text file and add your phrases there. When prompted by the game, enter the name of the file to open it.

All category files belong in the category folder.
